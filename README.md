## Project Setup
```sh
npm install
```

### Compile and Hot-Reload for Development
Create a `.env` file with `VITE_API_KEY=your_key` then run
```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
